##  Overview

<code>ShadcnUi-Dashboard-Demo is a demonstration dashboard application showing how to combine shadcn/ui components, Tailwind CSS, and React with Vite. It provides reusable UI building blocks, responsive layout, and example chart components for power and battery-related metrics. Use it as a starter template or a reference implementation for building admin panels and telemetry dashboards.</code>

---

##  Features

<code>
- Responsive dashboard layout with sidebar, header and content panes.
- Reusable UI components based on shadcn/ui (buttons, dialogs, inputs, etc.).
- Custom chart components for power and battery visualizations.
- Example pages for Dashboard, Power and Battery metrics.
- Fast development with Vite and Tailwind CSS.
- Data validation helpers with Zod and date handling with date-fns.
- Linting and code quality with ESLint.
</code>

##  Getting Started

###  Installation

Install ShadcnUi-Dashboard-Demo using one of the following methods:

**Build from source:**

1. Clone the ShadcnUi-Dashboard-Demo repository:
```sh
❯ git clone https://github.com/HimanthaThathsara/ShadcnUi-Dashboard-Demo
```

2. Navigate to the project directory:
```sh
❯ cd ShadcnUi-Dashboard-Demo
```

3. Install the project dependencies: **Using `npm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/npm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://www.npmjs.com/)

```sh
❯ npm install
```
